노드 목록 및 상세

특정 컬럼의 데이터가 표시되지 않는 것은 쿠버네티스 노드 에이전트가 구 버전이기 때문일 수 있습니다. 이런 경우 충분한 데이터가 수집될 수 있도록 쿠버네티스 에이전트 버전을 업데이트하세요.

노드 목록​

홈 화면 > 프로젝트 선택 > 리소스 > 노드 목록

현재 프로젝트의 노드 현황 정보를 테이블 형태로 확인할 수 있습니다.

제공하는 주요 정보는 다음과 같습니다.

분류주요 정보

에이전트 버전 등

다음 기능을 추가로 제공합니다.

  • 비교하기: 복수의 노드를 선택해 차트 기반으로 사용량을 비교할 수 있습니다.
  • CSV 다운로드: 화면에 표시된 목록 데이터를 CSV 파일로 다운로드할 수 있습니다.

서버 목록 페이지의 데이터는 60초 마다 자동 갱신됩니다. 사용자는 버튼을 클릭해 데이터의 자동 갱신을 중지하거나 버튼을 클릭해 자동 갱신을 재시작할 수 있습니다.

노드 상세​

홈 화면 > 프로젝트 선택 > 리소스 > 노드 상세

특정 노드의 상세 정보를 확인할 수 있습니다.

노드 상세 화면은 상단의 4개 차트와 노드 내 컨테이너 상세 정보로 구성되어 있습니다.

    • Network
    • Disk Usage
