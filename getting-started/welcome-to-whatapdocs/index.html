와탭에 오신 것을 환영합니다

와탭 모니터링 서비스는 다양한 플랫폼을 아우르는 통합 모니터링을 제공합니다. 수많은 디지털 소스에서 데이터를 수집해 사용자가 시스템과 시스템 개선 방안을 파악할 수 있도록 돕습니다.

  • 사용자는 와탭과 함께 다양한 데이터를 통합할 수 있습니다. 와탭 에이전트와 확장 도구를 활용해 데이터를 수집하고 하나의 UI로 사용자만의 디지털 생태계를 구축할 수 있습니다.
  • 사용자는 와탭과 함께 데이터를 분석할 수 있습니다. 수집한 데이터를 기반으로 사용자 정의형 대시보드 및 각종 차트 또는 강력한 쿼리 언어를 활용해 장애의 근본 원인을 찾고 시스템을 최적화할 수 있습니다.
  • 사용자는 와탭과 함께 시스템 장애에 신속하게 대응할 수 있습니다. 장애 징후를 먼저 감지하고 장애가 발생하기 전에 사용자에게 알릴 수 있습니다.

와탭 Docs 문서 안내​

와탭 Docs 문서는 사용자가 와탭 모니터링 서비스를 효과적으로 사용할 수 있도록 돕고자 합니다. 단계별 에이전트 설치부터 대시보드, 사용자 정의형 대시보드인 Flex 보드 설정, 각 성능 지표 정보, 알림 설정까지 와탭의 다양한 기능 활용 방법 및 권장 지침을 제공합니다.

Docs 홈 화면에서 상품별 문서 바로가기와 왼쪽 기본 메뉴를 확인할 수 있습니다.

  • 고정된 상단 메뉴를 통해 사용자는 와탭 Docs를 자유롭게 이동할 수 있습니다.
  • 아이콘을 선택하면 Docs 홈 화면으로 이동합니다.
  • 아이콘을 선택하면 와탭 Docs 페이지의 테마를 검은 배경으로 적용합니다.
  • 아이콘을 선택하면 와탭 Docs 페이지의 테마를 밝은 배경으로 적용합니다. 기본 적용 테마는 검은 배경입니다.
  • 상단 메뉴의 오른쪽에 있는 검색창을 통해 와탭 Docs 페이지에서 원하는 정보를 검색할 수 있습니다.
  • 서비스 화면 UI 텍스트는 파란색으로 표시됩니다.

시작하기​

시작하기 메뉴는 와탭 모니터링 서비스를 소개하고 플랫폼 경험과 기능 이해에 도움이 되도록 서비스 흐름을 안내합니다. 시작하기 메뉴를 선택하면 홈 화면으로 이동합니다. 지원 환경 문서에서 와탭 모니터링 서비스 상품별 지원 환경 개요를 확인할 수 있습니다. 홈 화면(UI) 안내 문서는 서비스 초기 화면 이용 방법을 안내합니다.

  • 와탭 모니터링 서비스에 대해 알고 싶다면 와탭 모니터링 서비스 안내 문서를 참조하세요.
  • 와탭 모니터링 서비스 이용이 처음이라면 빠른 시작하기 문서를 참조하세요.

상품별 문서 바로가기​

상품별 문서 바로가기 메뉴는 사용자가 선택한 상품에 대해 설치부터 설정 및 활용 지침까지 통합 안내 문서를 제공합니다.

  • 에이전트 설치 전 지원 환경을 확인하세요.
  • 설치 점검 사항을 확인하고 설치 중 문제가 발생한다면 설치 문제 해결 문서를 참조하세요.
  • 에이전트 설정 옵션과 업데이트부터 삭제 등의 관리 방법을 알아보세요.
  • 서비스 화면의 주요 메뉴 알아보기 문서에서 대시보드, Flex 보드, 보고서, 알림 설정 방법을 알아보세요.

각 상품별 문서는 Docs 홈 화면 왼쪽 기본 메뉴 혹은 고정된 상단의 상품별 문서 바로가기 메뉴에서 확인할 수 있습니다.

관리 기능​

관리 기능 메뉴는 와탭 모니터링 서비스의 계정, 프로젝트, 그룹 등을 관리하는 방법과 결제 방법을 안내합니다.

  • 보안이 강화된 멀티팩터 인증 방식을 확인하세요.
  • 와탭 모니터링 서비스의 모니터링 단위인 프로젝트의 계층 구조 및 관리 방법에 대해 알아보세요.
  • 와탭 모니터링 서비스의 무료 이용 기간 15일이 지났다면 결제 정보를 확인하고 유료 서비스 전환에 대해 알아보세요.

참조 문서​

참조 문서 메뉴는 와탭 Docs에서 사용하는 주요 개념 및 용어에 대한 자료집을 소개합니다. 메트릭스 정의, 와탭 메트릭스 조회를 위한 쿼리 언어 MXQL의 문법 및 명령어 안내, Open API 활용 문서 및 용어 사전을 제공합니다.

릴리스 노트​

릴리스 노트 메뉴에서 와탭 모니터링 서비스의 최신 기능 및 향상된 기능, 중요한 보안 패치에 관한 정보를 확인할 수 있습니다. 사용자 편의를 위해 서비스 릴리스 노트와 에이전트별 릴리스 노트를 구분하여 따로 제공합니다.

안내 외에 와탭 Docs와 관련한 궁금한 사항이나 추가 내용 요청 등의 피드백은 docs@whatap.io로 보내주세요.
