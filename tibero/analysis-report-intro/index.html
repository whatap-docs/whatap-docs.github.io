분석과 통계

분석하기​

수집한 정보를 바탕으로 특정 하루 동안의 전체적인 추이를 분석할 수 있는 기능을 제공하고 있습니다. 서로 다른 일자에 대한 추이 데이터를 비교해 데이터베이스의 성능 추이를 비교합니다. 특정 시점에 진행 중이던 세션을 파악해 느렸던 쿼리(Query)를 분석할 수 있습니다.

또한 하루 동안 발생한 Lock의 추이를 분석할 수 있습니다. Lock을 발생시킨 세션(Lock Holder)과 Lock을 대기하고 있는 세션(Lock Waitier)에 대해 트리 구조로 확인할 수 있습니다. 이러한 기능을 통해 특정 시점 발생한 락 정보 Holder와 Waiter 관계를 확인해 쿼리(Query)를 분석할 수 있습니다.

하루 이상의 장기 트렌드, 수집한 데이터의 Top N 쿼리를 분석할 수 있는 기능을 제공합니다.

통계 및 보고서​

수집한 SQL 통계를 통해 애플리케이션, 데이터베이스, 사용자, 호스트 단위로 통계를 낼 수 있습니다. 매 시간 저장한 테이블스페이스 및 데이터베이스 사이즈의 시간별 변화량을 확인할 수 있고 매일 저장한 테이블 사이즈의 변화량을 감지할 수 있습니다.

시간 단위 성능 지표 추이, 하루동안 발생한 SQL Top 10과 관련한 일간 보고서를 열람할 수 있습니다. 주간 보고서에서는 일 단위 성능 지표 추이 및 지난 주 성능 지표를 비교, 일주일 간의 SQL Top 10을 확인할 수 있습니다. 월간 보고서에서는 일 단위 성능 지표 추이, 1달간 발생한 SQL Top 10을 확인합니다.

기능 및 메뉴 안내​

와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 프로젝트 메뉴 하위 메뉴를 선택하세요.

  • 프로젝트 메뉴 > 분석 메뉴에서는 다음의 기능을 이용할 수 있습니다.

    • 카운트 추이
    • 카운트 추이 비교
    • 락 트리
    • 데드락
    • 데이터베이스 파라미터
    • 메트릭스 차트
    • 메트릭스 이상 탐지
    • 로그 파일 뷰어
    • 로그 파일 뷰어 (cloud)

  • 프로젝트 메뉴 > 통계/보고서

    통계/보고서 메뉴는 주간 및 일간과 월간 보고서, 그리고 통계를 제공합니다.
