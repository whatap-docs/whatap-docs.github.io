통합 메트릭스 보드

홈 화면 > 통합 메트릭스 보드

사용자가 속한 모든 프로젝트의 메트릭스 데이터를 조회할 수 있는 대시보드를 생성할 수 있습니다. 서로 다른 상품군의 메트릭스를 대시보드 위에 위젯으로 구성해 한눈에 파악할 수 있습니다.

통합 메트릭스 보드 메뉴를 처음 이용한다면 Flex 보드 메뉴를 사용할 것을 권장합니다. Flex 보드에 대한 자세한 내용은 각 상품별 문서를 참조하세요.

상품별 Flex 보드 문서
  • 애플리케이션 모니터링 > Flex 보드
  • 서버 모니터링 > Flex 보드
  • 데이터베이스 모니터링 > Flex 보드
  • 쿠버네티스 모니터링 > Flex 보드
  • Amazon ECS > Flex 보드
  • Amazon CloudWatch > Flex 보드
  • Azure Monitor > Flex 보드
  • Naver Cloud Monitoring > Flex 보드
  • Oracle Cloud Monitor > Flex 보드

위젯 차트 추가하기​

  1. 통합 메트릭스 보드 화면 위에 +위젯 추가를 선택하세요.

  2. 대시보드에 Dashboard 위젯이 생성됩니다. 위젯 오른쪽 위에 버튼을 선택하세요.

  3. 대시보드 생성하기 창이 나타납니다.

Series 차트 모양 설정하기​

차트 유형에서 Series를 선택하세요. Series 차트는 시간 기준으로 메트릭스 데이터의 흐름을 확인할 수 있습니다.

  • 미리보기: 설정한 옵션에 따라 차트의 모양이 변하는 것을 확인할 수 있습니다.

  • Title: 위젯 차트의 이름을 입력하세요.

  • X Axis

    • Axis Line: X축 선을 표시하거나 감출 수 있습니다.
    • Plot Line: 시간대별 수직선을 표시하거나 감출 수 있습니다.
    • Tick: X축 텍스트를 표시하거나 감출 수 있습니다.

  • Y Axis

    • Axis Line: Y축 선을 표시하거나 감출 수 있습니다.
    • Plot Line: 데이터별 수평선을 표시하거나 감출 수 있습니다.
    • Tick: Y축 텍스트를 표시하거나 감출 수 있습니다.
    • Inner Label: Y축 텍스트를 차트 안쪽으로 이동합니다.
    • uni: Y축 데이터의 단위를 설정할 수 있습니다.

  • Style

    • Area: 그래프 영역 안쪽을 색상을 채워 표시합니다.
    • Plot Verical Line
    • Show Plot Max Y Value: 차트에 가장 높은 수치의 값을 텍스트로 표시합니다.

  • Hover tooltip

    • Tick

Static 차트 모양 설정하기​

차트 유형에서 Static를 선택하세요. Static 차트는 메트릭스 데이터를 텍스트로 표현합니다. 실시간 데이터 값을 바로 확인할 때 용이합니다.

  • 미리보기: 설정한 옵션에 따라 차트의 모양이 변하는 것을 확인할 수 있습니다.

  • Title Style

    • Title: 메트릭스 차트 위젯의 제목을 입력하세요.
    • description: 메트릭스 차트 위제에 대한 설명을 입력하세요.
    • Title Font Color: 제목 색상을 설정합니다.

  • Content Label

    • Content Font Size: 차트에 표시할 텍스트의 크기를 설정합니다.
    • Content Background Color: 차트에 표시하는 텍스트의 배경 색상을 설정합니다.
    • Content Front Color: 차트에 표시할 텍스트의 색상을 설정합니다.
    • Vertical Align: 텍스트의 수직 정렬을 설정합니다.
    • Horizontal Align: 텍스트의 수평 정렬을 설정합니다.

  • Formats

    • format: 메트릭스 데이터의 단위를 설정합니다.
    • Separator: 1000단위를 쉼표(,)로 구분할 수 있습니다.
    • Prefix: 차트에 표시하는 텍스트 앞에 접두어를 붙일 수 있습니다.
    • Suffix: 차트에 표시하는 텍스트 뒤에 접미어를 붙일 수 있습니다.

Color alert 차트 모양 설정하기​

차트 유형에서 Color alert을 선택하세요. Color alert 차트는 메트릭스 데이터의 지연 구간을 파악하는데 용이합니다. 기본 3단계로 구간을 설정할 수 있으며, 빼거나 더할 수 있습니다.

  • Title: 차트의 제목을 입력하세요.

  • Values

    • Name: 구간마다 이름을 입력해 현재 상태를 정의할 수 있습니다.
    • Min: 지연되는 시간을 분 단위로 설정할 수 있습니다.
    • Color: 지연 구간마다 색상을 설정해 직관적으로 현황을 파악할 수 있습니다.

    불필요한 단계를 삭제하려면 버튼을 선택하세요. 단계을 추가하려면 + Add 버튼을 선택하세요.

기준 시간 설정하기​

데이터 설정: 설정한 시간을 기준으로 차트에 데이터를 표시합니다.

  • 실시간: 최근 5분, 10분, 30분, 1시간, 3시간, 6시간, 12시간을 기준으로 실시간 데이터를 확인할 수 있습니다.
  • 과거: 오늘, 어제, 이틀 전, 3일 전, 일주일 전을 기준으로 최근 5분 데이터를 확인할 수 있습니다. 오늘 이전을 선택하면 현재 시간 기준으로 5분 데이터를 확인할 수 있습니다.
  • 커스텀: 원하는 날짜와 시간을 선택해 시간을 설정할 수 있습니다.

태그 데이터 추가하기​

태그 데이터 추가 버튼을 선택해 메트릭스 데이터를 불러올 프로젝트를 선택하세요.

  • 버튼을 선택해 태그 데이터의 이름을 변경할 수 있습니다.

  • 시간: 데이터 설정에서 설정한 시간을 기준으로 데이터를 불러와 차트를 그리거나 이전 시간과 비교해 차트를 그릴 수 있도록 설정할 수 있습니다.

  • 프로젝트: 상품별 프로젝트를 선택할 수 있습니다. 프로젝트를 선택하면 카테고리 항목이 나타납니다.

  • 카테고리: 선택한 프로젝트에서 메트릭스를 불러올 카테고리를 선택하세요. 카테고리를 선택하면 아래에 추가 옵션이 나타납니다.

  • 태그: 수집 대상을 선택할 수 있습니다. 에이전트 이름(Oname), 그룹(OkindName), 호스트 IP(Hostip), 형태(Type)별로 선택할 수 있습니다.

  • 필드: 차트에 표시할 메트릭스를 선택하세요.

  • 시간 병합: 원본 데이터에서 필드 값이 같은 데이터끼리 시간 범위에 따라 일정한 시간 간격(5분 또는 1시간)으로 데이터를 병합합니다.

  • 오브젝트 병합: 태그 항목에서 선택한 값을 기준으로 데이터를 그룹화할 때 사용합니다. 서로 다른 필드값을 가진 데이터들 중에서 태그가 일치하는 경우 해당 데이터를 병합하는 방법입니다.

모든 항목에 대한 설정을 마쳤으면 완료 버튼을 선택하세요.

  • 선택한 프로젝트에 따라 카테고리에서 선택할 수 있는 항목은 다를 수 있습니다.
  • 메트릭스에 대한 자세한 내용은 다음 문서를 참조하세요.
  • 시간 병합 및 오브젝트 병합에 대한 자세한 내용은 다음 문서를 참조하세요.

위젯 차트 확인하기​

위젯 차트를 추가한 다음 실시간으로 메트릭스의 추이를 확인하려면 화면 상단에 뷰 모드를 선택하세요. 위젯 차트를 추가하거나 수정, 삭제하려면 수정 모드를 선택하세요.

위젯 차트 수정하기​

위젯 차트를 수정하려면 수정 모드로 전환한 다음 위젯 차트 오른쪽 위에 버튼을 선택하세요. 대시보드 생성하기 창이 나타나면 원하는 항목을 수정할 수 있습니다. 수정을 완료하면 확인 버튼을 선택하세요.

위젯 차트 복사하기​

같은 유형의 위젯 차트에서 메트릭스 값만 변경해 사용하려면 위젯 차트를 복사해 사용할 수 있습니다. 수정 모드로 전환한 다음 위젯 차트 오른쪽 위에 버튼을 선택하세요. 메트릭스를 변경하려면 위젯 차트 오른쪽 위에 버튼을 선택한 다음 원하는 항목을 수정하세요.

위젯 차트 삭제하기​

위젯 차트를 삭제하려면 수정 모드로 전환한 다음 위젯 차트 오른쪽 위에 버튼을 선택하세요. 팝업 창이 나타나면 확인 버튼을 선택하세요.
