Flex 보드 공유하기

Flex 보드를 다른 계정의 사용자와 공유하거나 재사용할 수 있습니다.

구성원과 Flex 보드 공유하기​

사용자가 생성한 Flex 보드를 다른 사용자와 공유해 동일한 대시보드를 볼 수 있습니다. 이를 통해 업무를 공유할 수 있습니다. 공유한 대시보드를 수정한 경우 공유 받은 모든 사용자는 동일한 대시보드를 확인할 수 있습니다. 이 기능은 홈 화면 > Flex 보드 메뉴에서만 이용할 수 있습니다.

  1. Flex 보드 메뉴의 대시보드 목록에서 공유하려는 대시보드의 버튼을 선택하세요.

  2. 대시보드 공유하기 창이 나타나면 + 계정 버튼을 선택하세요.

  3. 대시보드를 공유할 사용자 계정을 선택하세요.

    • 공유 대상을 추가하려면 + 계정 버튼을 선택하세요. 사용자 계정을 추가로 선택할 수 있습니다.
    • 대시보드를 공유할 수 있는 사용자 계정은 선택한 프로젝트에 소속된 사용자 계정입니다.
    • 대시보드의 모든 프로젝트에 대한 수정 권한이 없는 경우 읽기 전용 태그가 나타납니다.

  4. 대시보드 공유하기 버튼을 선택하세요. 읽기 전용 체크박스를 선택하면 공유 받은 사용자는 대시보드를 수정할 수 없습니다.

대시보드를 공유한 항목은 대시보드 목록에서 공유 태그가 표시된 것을 확인할 수 있습니다. 공유 태그에 마우스를 오버하면 대시보드를 공유한 구성원에 대한 정보를 확인할 수 있습니다.

대시보드를 공유 받은 경우대시보드를 공유한 경우(대시보드 소유자)

대시보드 공유 조건

  • 대시보드 공유 기능은 홈 화면 > Flex 보드 메뉴에서만 이용할 수 있습니다.

  • 대시보드를 공유하려는 멤버는 대시보드의 모든 프로젝트에 대한 수정 권한을 가지고 있어야 합니다.

  • 모든 프로젝트에 대한 수정 권한이 없는 상태에서 대시보드를 공유하면 읽기 전용 모드로 공유합니다. 권한에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 공유 받은 대시보드를 다른 사용자에게 다시 공유할 경우 해당 대시보드의 속성은 읽기 전용 모드가 아니어야 합니다.

  • 공유 조건에 해당하는 경우에만 대시보드 공유 버튼이 노출됩니다.

  • 대시보드 소유자가 대시보드를 삭제하는 경우 대시보드를 공유한 모든 사용자 계정에서도 해당 대시보드를 삭제합니다.
  • 대시보드를 공유 받은 사용자가 대시보드를 삭제하면, 해당 사용자의 대시보드 목록에서만 삭제합니다. 원본 대시보드는 그대로 유지합니다.

Flex 보드 복사하기​

사용자가 생성한 Flex 보드를 복사해 다른 사용자에게 전달할 수 있습니다.

Flex 보드 메뉴의 보드 목록에서 버튼을 선택하세요. 나에게 복사 또는 다른 사람에게 복사 옵션을 선택하세요.

  • 나에게 복사 옵션을 선택하면 보드 목록에 '_copied' 접두어가 붙은 항목이 추가됩니다.

  • 다른 사람에게 복사 옵션을 선택하면 + 계정을 클릭하세요. 다른 사용자 계정의 이메일을 입력한 다음 대시보드 복사하기 버튼을 선택하세요. 읽기 전용 옵션을 선택하면 공유 받은 사용자가 대시보드를 수정할 수 없습니다.

프로젝트 메뉴 > Flex 보드 메뉴에서 복사 기능은 프로젝트 멤버와 대시보드를 공유합니다.

Flex 보드 내보내기, 가져오기​

json 파일로 저장한 Flex 보드 설정을 다른 사용자 계정에서 가져와 복사할 수 있습니다. 대시보드 관리자 권한의 기능입니다.

  1. Flex 보드 메뉴의 보드 목록에서 버튼을 선택하세요.

  2. 웹 브라우저에서 Flex 보드 설정을 json 파일로 다운로드합니다.

  3. 화면 오른쪽 위에 가져오기 버튼을 선택하세요.

  4. json 파일을 선택하세요.

이 기능은 프로젝트 수정 권한을 소유한 사용자만 이용할 수 있습니다.
