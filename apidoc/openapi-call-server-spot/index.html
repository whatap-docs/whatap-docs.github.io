활성화 상태의 에이전트 수​

GET https://service.whatap.io/open/json/act_agent
command
$ curl -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://service.whatap.io/open/json/act_agent"
서버 모니터링 상품군 response
{
  "pname": "LOCAL_INFRA",
  "values": "57.0"
}
