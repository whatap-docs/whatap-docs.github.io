인스턴스 모니터링

홈 화면 > 프로젝트 선택 > 대시보드 > 인스턴스 모니터링

데이터베이스 서버의 핵심 지표, 액티브 세션 및 락 트리(Lock Tree)에 대한 상세 정보를 실시간으로 모니터링 할 수 있습니다. 또한 문제가 될 수 있는 경고들을 확인할 수 있습니다. 수집 데이터는 5초 간격으로 갱신합니다.

조회 시간 및 대상 선택하기​

기본적으로 지난 10분간의 수집 데이터를 실시간으로 조회합니다. 시간과 인스턴스 항목에서 원하는 시간과 대상을 선택하면 수집 데이터가 그래프 차트와 Active Sessions 테이블에 자동 반영됩니다.

  • 과거 데이터를 조회하려면 녹색 버튼을 클릭한 다음 원하는 조회 시간을 선택하세요. 최대 지난 3주까지의 데이터를 조회할 수 있습니다.

  • 임의의 날짜와 시간을 조회하려면 시계 표시 부분을 선택하세요. 날짜와 시간 텍스트 영역을 클릭하면 날짜와 시간을 선택할 수 있는 옵션이 나타납니다. 다시 실시간으로 데이터를 조회하려면 버튼을 선택하세요.

  • 다른 인스턴스의 데이터를 조회하려면 인스턴스를 클릭한 다음 원하는 데이터베이스 서버의 이름을 선택하세요.

그래프 차트 위젯​

Default 프리셋으로 저장한 16가지 지표를 그래프 차트 형식의 위젯으로 표시합니다. 가로축은 시간, 세로축은 각 지표의 수치 값입니다. 각 지표의 의미를 알고 싶다면 이름 옆에 i 버튼을 선택하거나 다음 문서를 참조하세요.

  • 위젯의 위치를 변경하려면 위젯의 윗 부분을 선택한 상태에서 드래그하세요.

    위젯의 크기는 변경할 수 없습니다.

  • 위젯에서 특정 시간을 선택하면 실시간 조회 모드를 중지하고 선택한 시간에 수행한 액티브 세션 정보를 Active Sessions 테이블에 표시합니다.

지표 선택하기​

위젯에 지표 그래프를 추가하거나 교체하려면 위젯의 오른쪽 위에 버튼을 선택하세요. Metric Select 창이 나타납니다. 원하는 설정을 완료한 다음 저장 버튼을 선택하세요.

  • 미리보기: 선택한 지표에 대한 그래프를 미리 볼 수 있습니다.

  • 제목: 위젯의 이름을 입력하세요.

  • 추천 지표: 토글 버튼을 선택하면 추천 지표 또는 모든 지표를 확인할 수 있습니다. 추천 지표는 데이터베이스 모니터링에서 자주보는 지표들을 모아놓은 목록입니다.

  • 영역에서 추가할 지표를 검색한 다음 해당 지표 오른쪽에 버튼을 선택하세요. 선택한 지표는 영역에 추가됩니다.

  • 영역에서 지표의 추이를 그래프로 확인하고 위젯을 추가할 수도 있습니다. 위젯의 오른쪽 위에 버튼을 선택하세요. Cateogry에서 원하는 분류를 선택해 지표 목록을 간추릴 수 있습니다.

  • 추가한 지표를 다른 지표로 변경하려면 영역에서 변경하려는 지표를 선택하세요. 아래 지표 목록에서 변경하려는 지표의 이름 또는 위젯을 선택하세요.

  • 추가한 지표를 삭제하려면 영역에서 삭제하려는 항목의 버튼을 선택하세요. 지표는 적어도 하나 이상 선택해야 합니다.

프리셋 저장​

위젯의 지표를 추가 또는 변경하고 레이아웃 상태를 프리셋으로 저장하고 불러올 수 있습니다. Default 프리셋은 수정할 수 없습니다.

  1. 위젯의 지표를 추가 또는 변경하세요.

  2. 화면 오른쪽 위에 프리셋 저장 버튼을 선택하세요.

  3. 이름을 변경한 다음 저장 버튼을 선택하세요.

프리셋 목록에서 저장한 항목을 선택해 설정을 불러올 수 있습니다.

액티브 세션 / 락 트리​

실시간 수행 중인 액티브 세션 및 락 트리 정보를 조회할 수 있습니다. 위젯에서 특정 시간을 선택하면 실시간 조회 모드를 중지하고 선택한 시간에 수행한 세션 정보를 테이블에 표시합니다. 테이블 위에 5초 단위로 데이터를 조회할 수 있는 시간 선택 버튼이 나타납니다.

  • Acitive Sessions

    실시간 수행 중인 세션과 쿼리 내용을 조회할 수 있습니다. 테이블에서 가장 오른쪽에 query 컬럼의 항목을 선택하면 SQL 보기 창이 나타납니다. 선택한 쿼리(Query)에 대한 Full Text 및 Plan 정보를 확인할 수 있습니다. 테이블의 컬럼에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 락 트리

    실시간으로 락 세션에 대한 정보와 Holder, Waiter 정보를 조회할 수 있습니다. 테이블의 컬럼에 대한 자세한 내용은 다음 문서를 참조하세요.

데이터 목록 필터링​

테이블에 있는 데이터를 검색어를 입력해 필터링할 수 있습니다.

  1. 테이블 오른쪽 위에 버튼을 선택하세요.

  2. 컬럼 항목과 조건을 선택하고 검색어를 입력하세요.

  3. 저장 버튼을 선택하세요.

저장한 내용은 버튼 옆에 표시됩니다.

  • 저장한 조건을 선택해 필터링을 적용하거나 해제할 수 있습니다.
  • 추가 필터링 조건을 만들어 중복 적용할 수 있습니다.
  • 필터링 조건을 삭제하려면 버튼을 선택하세요.

테이블 컬럼 변경하기​

테이블의 컬럼 항목을 표시하거나 숨길 수 있고, 순서를 변경할 수 있습니다.

  1. 테이블의 오른쪽 위에 버튼을 선택하세요.

  2. 컬럼 편집 창이 나타나면 테이블의 컬럼에 표시할 항목을 숨기거나 추가, 순서를 변경하세요.

    • 컬럼 항목을 선택한 다음 또는 , 또는 버튼을 선택하세요. 표시 또는 숨김으로 이동합니다.
    • 컬럼 항목을 선택한 상태에서 드래그하면 위치를 이동할 수 있습니다.

  3. 설정을 완료한 다음 저장 버튼을 선택하세요.

설정한 내용은 브라우저 쿠키값으로 저장되어 페이지 새로 고침 후에도 상태를 유지합니다. 쿠키 삭제 또는 기타 이유로 설정 간 오류가 발생하면 초기화합니다.
